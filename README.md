# 🌐 ViewBrowser - Advanced Web Proxy & VPN Browser

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![.NET](https://img.shields.io/badge/.NET-8.0-blue.svg)](https://dotnet.microsoft.com/)
[![Live Demo](https://img.shields.io/badge/Demo-mybrowser.ir-green.svg)](https://mybrowser.ir)

An advanced web proxy and VPN browser built with ASP.NET Core that allows secure, encrypted browsing through a VPN tunnel with comprehensive content rewriting and session management.

## 🚀 Features

### 🔒 Security & Privacy
- **VPN Session Management**: Secure, encrypted VPN tunnels for each browsing session
- **End-to-End Encryption**: All traffic is encrypted using session-specific keys
- **DNS over HTTPS**: Enhanced privacy with DoH support
- **IP Masking**: Browse anonymously with IP address protection

### 🌐 Advanced Proxy Capabilities
- **URL Rewriting**: Intelligent rewriting of HTML, CSS, and JavaScript resources
- **Content Processing**: Automatic processing of images, videos, scripts, and stylesheets
- **Form Handling**: Seamless form submission through proxy
- **Cookie Management**: Session-based cookie storage and forwarding
- **Redirect Following**: Automatic handling of HTTP redirects

### ⚡ Performance
- **Smart Caching**: Configurable caching system for faster page loads
- **Compression Support**: GZip, Deflate, and Brotli compression
- **Concurrent Connections**: Support for multiple simultaneous requests
- **Response Time Tracking**: Real-time performance metrics

### 🛡️ Content Control
- **Domain Filtering**: Whitelist/blacklist support for allowed domains
- **Content Type Detection**: Intelligent handling of different content types
- **Binary Content Support**: Proper handling of images, videos, and other binary files
- **CORS Handling**: Cross-origin resource sharing management

## 📋 Prerequisites

- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) or higher
- Visual Studio 2022 or Visual Studio Code
- Modern web browser (Chrome, Firefox, Edge, Safari)
